# IDE Updater

A CLI tool to quickly update Linux IDEs (VS Code, Cursor, Kiro) with a simple command.

## Usage

```bash
# Update all IDEs
ide-updater update

# Update specific IDE
ide-updater update cursor
ide-updater update vscode
ide-updater update kiro

# Check status
ide-updater check
```

## Installation

```bash
pip install -e .
```
